# hyperpay_sdk_ios

Vendored XCFrameworks for the HyperPay (OPPWA) Mobile SDK v7.4.0.

This pod is consumed as a dependency of [`hyperpay_sdk`](https://pub.dev/packages/hyperpay_sdk) and is not intended for direct use.

## Frameworks Included

- `OPPWAMobile.xcframework` — HyperPay OPPWA Mobile SDK
- `ipworks3ds_sdk_deploy_9373.xcframework` — 3DS2 authentication (release)
- `ipworks3ds_sdk_9373.xcframework` — 3DS2 authentication (debug/simulator)
